# PhotoTool

## Important
This project is made for Windows and therefore only works there.

## Dependencies
To install the required packages, run ```pip install -r requirements.txt``` or create a virtual environment. 
<br>You need to have any version of poppler installed (tested with 25.12.0-0). You can find it here: https://github.com/oschwartz10612/poppler-windows. The ```bin\``` directory has to be added to PATH.
<br>It includes pdf2image, Pillow, pynput, pypdf, screeninfo and watchdog. Newer versions than listed in the file may work, but there is no guarantee. 
<br>The project was tested with Python 3.10.7.

## Run the Application
Simply run ```main.pyc``` with any supported version of Python.
